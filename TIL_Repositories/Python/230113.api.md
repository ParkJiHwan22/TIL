## API(Application Programming Interface)
- 컴퓨터나 컴퓨터 프로그램 사이의 연결
- 일종의 소프트웨어 인터페이스 이며 다른 종류의 소프트웨어에 서비스를 제공
사용하는 방법을 기술하는 문자나 표준은 API 사양/명세 (specifcation)
